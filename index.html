❓ Guia de Uso — Ornitologia Avançada de Santa Catarina

Este sistema permite importar listas de aves, visualizar dados taxonômicos e de conservação, gerar árvores filogenéticas e mapear ocorrências georreferenciadas com dados do GBIF e eBird. Tudo roda no seu navegador — nenhum dado é enviado a servidores externos.

1. 📥 Importação de Dados

A importação é o ponto de partida. Você insere nomes de aves e o sistema preenche automaticamente os campos taxonômicos.

Modo Básico (padrão)

  • Digite o nome científico ou popular de uma ave no campo de texto. Sugestões aparecem enquanto você digita.
  • Use ↑ ↓ para navegar nas sugestões e Tab ou Enter para confirmar.
  • Para importar várias de uma vez, cole uma lista (um nome por linha) e clique em Processar Dados Importados.
  • Filo, Classe, Ordem, Família e Subfamília são preenchidos automaticamente pelo banco de dados interno (>800 espécies).

Modo Avançado

  • Para dados já tabulados. Cole no formato: Filo↹Classe↹Ordem↹Família↹Subfamília↹Gênero/Espécie (tabulação como separador).
  • Clique em Processar Dados Importados.

Botões auxiliares

  • Carregar Exemplo: insere 5–10 espécies aleatórias para teste rápido.
  • Fontes de Estudo: exibe as referências científicas usadas na filogenia.
  • Comparar Tabelas: abre o comparador de listas (ver seção 7).
  • Lista de Consulta: abre o banco com todas as espécies cadastradas (ver seção 8).

2. 📋 Tabela de Dados

Após importar, os dados ficam numa tabela editável abaixo da área de importação (seção Dados da Tabela).

  • Clique em qualquer célula para editar. A árvore filogenética se atualiza automaticamente.
  • Adicionar Linha Manualmente: insere linha em branco.
  • Remover: botão ao lado de cada linha.
  • Limpar Tabela: remove tudo.
  • Copiar Tabela: copia o conteúdo para a área de transferência (formato TSV, compatível com Excel).

💡 No modo básico, o autocomplete também funciona dentro do campo Gênero/Espécie da tabela.

3. 🌳 Árvore Taxonômica

Gerada automaticamente a partir dos dados da tabela.

  • Árvore Hierárquica: estrutura de pastas aninhadas por nível taxonômico.
  • Árvore Filogenética (D3.js): diagrama interativo baseado na filogenia moderna das aves (Jarvis 2014, Prum 2015).
    • Zoom: botões + / − ou scroll do mouse.
    • Arrastar: clique e arraste no fundo da visualização.
    • Mostrar Completa vs Apenas Adicionados.
  • Baixar Imagem: exporta PNG de alta qualidade.
  • Exportar Árvore: salva HTML interativo com zoom e busca.

4. 🛡️ Status de Conservação

Dados das listas de ameaça SC (Santa Catarina), ICMBio e IUCN.

  • Clique em Carregar Dados do PDF (SC) e WikiAves (IUCN) para preencher a tabela com as espécies presentes.
  • Ordene pela coluna desejada usando os botões A-Z, Conservação ▲ e Conservação ▼.
  • Mais informações: links WikiAves por espécie.
  • Exportar Excel (CSV) e Copiar Tabela disponíveis.
  • Três gráficos de pizza mostram a distribuição percentual dos status para SC, ICMBio e IUCN.

5. 📊 Distribuição Taxonômica

Gráficos automáticos de barras e pizza por Ordem e por Família, refletindo sempre os dados atuais da tabela.

6. 🗺️ Mapa de Ocorrências

Ferramenta central para análise espacial. Permite posicionar espécies no mapa, desenhar áreas de estudo e buscar registros públicos georreferenciados.

Painel "Espécies na tabela"

  • Lista todas as espécies importadas. Cada item tem:
    • Clique no nome → pisca os pontos da espécie em amarelo 2× no mapa (funciona para pontos manuais, GBIF e eBird).
    • Botão 👁️ (canto direito) → oculta ou exibe todos os pontos daquela espécie no mapa.
  • Arraste espécies do painel para o mapa para posicioná-las manualmente.

Desenhar áreas

  • 🔷 Desenhar área: clique no mapa para criar vértices de um polígono. Dê duplo-clique para fechar.
  • ➕ Adicionar todas: posiciona automaticamente todas as espécies da tabela dentro do polígono desenhado (pontos aleatórios dentro do perímetro).
  • ✏️ Editar área: ativa handles de edição nos polígonos já desenhados.
  • 📁 Carregar Shapefile (ZIP): importa polígonos de um arquivo ZIP com shapefile.
  • 🔢 Calcular espécies nas áreas: gera uma tabela com a contagem de espécies, pontos, área e densidades para cada polígono desenhado.
  • 🗺️ Exportar Shapefile: baixa um ZIP com os pontos e polígonos do mapa em formato Shapefile (.shp/.dbf/.prj).

Pontos manuais

  • ✏️ Edição de ponto: ativa o modo arrasta-e-solta nos marcadores.
  • 🗑️ Lixeira: quando ON, clicar em um ponto o remove do mapa.

Registros públicos de ocorrência

  • 🌍 GBIF: busca ocorrências georreferenciadas no Brasil a partir do banco global GBIF (até 2000 registros por espécie). Os dados são carregados uma vez por sessão.
  • 🐦 eBird: busca especificamente registros da plataforma eBird (via GBIF), focada em observações de campo de birders. Até 2000 registros por espécie. Também carregado apenas uma vez por sessão — desligar e religar apenas esconde/exibe sem rebuscar.

7. 🔀 Comparador de Listas

Acesse pelo botão Comparar Tabelas. Permite comparar múltiplas listas de espécies side-by-side.

  • Adicione quantas listas quiser. Em cada uma: cole os nomes, defina a legenda e clique em Processar Lista.
  • Clique em Executar Comparação para gerar a tabela e os gráficos (espécies por lista, similaridade de Jaccard, espécies únicas).
  • Filtros: Todas, ✅ Presentes em todas, ❌ Ausentes de alguma.
  • Exportar CSV e Copiar Tabela disponíveis.

8. 🔍 Consulta de Espécies

Botão Lista de Consulta — exibe o banco completo com mais de 800 espécies de aves do sul do Brasil.

  • Filtre pelo campo de busca.
  • Clique em qualquer linha para adicionar a espécie à tabela principal.
  • Links diretos para o WikiAves de cada espécie.

9. 💡 Dicas Gerais

  • Todas as seções são colapsáveis — clique no cabeçalho verde (ícone −/+) para recolher/expandir.
  • As categorias IUCN usadas são: LC (pouco preocupante), NT (quase ameaçada), VU (vulnerável), EN (em perigo), CR (criticamente em perigo), DD (dados insuficientes), NE (não avaliada).
  • O sistema funciona 100% offline após o carregamento (exceto buscas GBIF/eBird que precisam de internet).
  • Use um navegador atualizado (Chrome ou Firefox recomendados) para melhor desempenho.
  • Nenhum dado é salvo no servidor — ao recarregar a página os dados são perdidos. Use Copiar Tabela para guardar seu trabalho.
